1.header tag vs h1 tag ?

A.

header Tag h1 Tag
  • header element represtns a container for introductory content or a set of navigational links.
  • A header element typically contains:
    • one or more heading elements(h1-h6)
    • logo or icon
    • authorship information
  • we can have several header element in one HTML document. However,header cannot be placed within a footer, address or another header element
  • The h1 to h6 tags are used to define HTML headings.
  • h1 defines the most important heading. h6 defines the least important heading
  • Only Use one h1 per page-this should represent the main heading/subject for the whole page

6.What is the difference between main vs body tag ?

A. main Tag:-

  • main tag specifies the main content of a document.
  • The content inside the main element should be unique to the document.
  • It should not contain any content that is repeated across documents such as sidebars, navigation links, copyright information, site logos, and search forms.
  • There must be only one main element in a document & We should not add main tag under article, aside, footer, header or nav.
body Tag:-
  • body tag defines the document's body.
  • the body element contains all the contents of an HTML document, such as headings, paragraphs, image, hyperlinks, tables, list etc.
  • There can only be one body element in an HTML document
